# Office Essentials Collection

## Collection Information
**Name:** Office Essentials
**Purpose:** Professional office supplies and accessories for modern businesses

## Products in This Collection

This collection contains professional office products. Each product folder includes:
- Complete product details in the `Details/` subfolder
- High-quality product images in the `Photos/` subfolder

## Adding a New Product to This Collection

1. Create a new folder with the product name
2. Inside the folder, create two subfolders:
   - `Details/` - Contains all product information files
   - `Photos/` - Contains product images

3. In the `Details/` folder, create these required files:
   - `Name.txt` - Product display name
   - `Description.txt` - Detailed product description
   - `SKU.txt` - Unique SKU code
   - `ItemCost.txt` - Price for a single item (decimal format: 12.50)
   - `BoxCost.txt` - Price for a box (decimal format: 120.00)
   - `UnitsPerBox.txt` - Number of items per box (integer: 12)

4. In the `Photos/` folder, add images:
   - `main.png` or `main.jpg` - Primary product image (shown first)
   - Additional images with any descriptive names
