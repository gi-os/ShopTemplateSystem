# Public Assets

This directory contains publicly accessible assets.

## Easter Egg Cat Photo

The file `basil-cat.jpg` should contain the adorable photo of Basil the cat wearing a tie.
This photo is displayed on the secret easter egg page at `/secret`.

To add the photo:
1. Save the cat photo as `basil-cat.jpg` in this directory
2. The image will be automatically served at `/basil-cat.jpg`
